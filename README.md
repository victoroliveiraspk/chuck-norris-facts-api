# Chuck Norris facts API

## Technologies that was used to build this app:

```
NestJS
Typescript
GraphQL
Jest
```

## How to set up:

### To make local server, run:

```
yarn start:dev
```

### To make the build direcotry, run:

```
yarn build
```

### To make local build server, run:

```
yarn build
yarn start:prod
```

### To run tests:

```
yarn test:e2e
